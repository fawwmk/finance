# Expo

Проект на **Expo SDK 56**, не 57: 57 несовместим с Expo Go на этом телефоне.

Читай документацию именно по своей версии: https://docs.expo.dev/versions/v56.0.0/
(Раньше здесь была ссылка на 57 — из-за неё в проект заехал `expo-font@57`
рядом с `expo-font@56`, а две версии одного нативного модуля ломают сборку.)

Пакеты ставь через `npx expo install`, а не `npm install` — только так подберётся
версия, совместимая с SDK.

## Это приложение считает чужие деньги

Ошибка в формуле здесь — не баг, а потерянные деньги живого человека.

- Денежную функцию проверяй на реальных числах, а не «на глаз»: скомпилируй
  модуль отдельно (`./node_modules/.bin/tsc <файл> --ignoreConfig --outDir …`)
  и прогони сценарий в node.
- Нет курса валюты — сумму НЕЛЬЗЯ ни показывать, ни складывать. Никаких
  «подставим один к одному»: 500 € — это не 500 ₽.
- Ответ модели, HTML чужого сайта, удалённый конфиг — это НЕ доверенные данные.
  Проверяй тип и диапазон каждого числа, прежде чем пускать его в расчёт.
